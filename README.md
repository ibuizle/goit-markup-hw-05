# Layout with Animations & Modals 🎬

This repository contains the fifth homework assignment for the GoIT Markup course. The project focuses on adding interactivity to a web page using **CSS transitions, animations**, and building a complex **modal window**.

🔗 **[Live Demo](https://ibuizle.github.io/goit-markup-hw-05/)**

## 📖 About the Project

The goal of this assignment was to enhance the user experience (UX) by adding smooth visual effects and interactive elements. It covers the creation of a backdrop, a centered modal window with a contact form, and various hover/focus effects for buttons and links.

### Key Learnings & Features:
- **CSS Transitions:** Creating smooth changes for properties like `color`, `background-color`, and `transform`.
- **CSS Animations:** Implementing keyframe animations for more complex visual movements.
- **Modal Window:** Building a hidden modal that appears over the content with a darkened backdrop.
- **Form Styling:** Designing an interactive contact form inside the modal with custom input styles and icons.
- **SVG Icons:** Using vector graphics to create sharp, scalable icons for links and form fields.
- **Positioning:** Mastered the use of `position: fixed` for the backdrop and `position: absolute` for centering the modal.

## 🛠 Built With

- **HTML5:** Semantic markup for a structured and accessible web page.
- **CSS3:** Advanced styling using Flexbox, transitions, and custom animations.
- **SVG:** Scalable Vector Graphics for high-quality icons.
